# Alke Wallet

Proyecto de evaluación para el Módulo 2. Es una simulación de una billetera virtual (E-wallet) desarrollada con tecnologías web estándar.

## Descripción

Esta aplicación web permite a los usuarios gestionar flujos básicos de dinero. El objetivo principal fue crear una interfaz amigable y funcional que simule las operaciones reales de una banco digital, aplicando conceptos de maquetación y lógica de programación.

## Funcionalidades

- **Autenticación**: Sistema de login básico con validación.
- **Depósitos**: Permite sumar fondos al saldo actual.
- **Transferencias**: Envío de dinero a contactos predefinidos con validación de saldo.
- **Historial**: Visualización dinámica de los últimos movimientos.
- **Diseño Responsivo**: Adaptable a diferentes tamaños de pantalla.

## Tecnologías Utilizadas

- **HTML**: Estructura semántica.
- **CSS**: Estilos personalizados y paleta de colores monocromática (Azul).
- **Bootstrap**: Framework para la grilla y componentes UI.
- **JavaScript**: Lógica de la aplicación y manipulación del DOM.
